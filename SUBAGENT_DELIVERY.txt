📊 FIRESTORE SCHEMA (4 COLLECTIONS)
================================================================================

ddhg_users
──────────
├── userId (Primary Key)
├── totalDDC (current balance)
├── plantGrowthStage (0-6)
├── streakCount (global streak)
├── habits (array of habit IDs)
├── lastCompletedAt (timestamp)
├── created_at / updated_at

habit_completions
─────────────────
├── userId (FK)
├── habitId
├── habitName
├── source (habit-garden|dishrated|trainlog|planning|gratitude)
├── completedAt (timestamp)
├── ddcEarned (amount)
├── streakIncremented (boolean)
├── growthStageIncremented (boolean)
├── mileachieved (3day|7day|30day)
├── notes
├── created_at

ddc_ledger (transaction log)
────────────────────────────
├── userId (FK)
├── amount
├── reason (habit_complete|streak_milestone)
├── habitId (FK)
├── transactionType (earn|bonus)
├── balanceBefore / balanceAfter
├── created_at

growth_milestones
─────────────────
├── userId (FK)
├── milestone (3day|7day|30day)
├── habitId (FK)
├── achievedAt (timestamp)
├── ddcBonus
├── created_at

🚀 HOW TO USE (PHASE 2 INTEGRATION)
================================================================================

RECORD A HABIT COMPLETION:
──────────────────────────
import { saveHabitCompletion } from '@/lib/habitFunctions'

await saveHabitCompletion(
  userId,
  'meditation',           // habit ID
  'Morning session',      // notes (optional)
  '09:30',                // time (optional)
  'habit-garden'          // source app
)

Automatically:
  ✅ Saves to Firestore
  ✅ Awards 1 DDC
  ✅ Increments streak
  ✅ Updates plant growth stage

LOAD USER DATA:
───────────────
import { loadUserHabits } from '@/lib/habitFunctions'

const habits = await loadUserHabits(userId)

Returns:
  {
    completedToday: ['meditation', 'breakfast'],
    totalDDC: 45,
    currentStreak: 14,
    plantGrowthStage: 3,
    streaks: { meditation: 14, breakfast: 8, ... }
  }

SUBSCRIBE TO REAL-TIME UPDATES:
───────────────────────────────
import { subscribeToCompletions } from '@/lib/integrations'

subscribeToCompletions(userId, (completions) => {
  // Triggered when new completions are recorded

  // Trigger growth animation
  // Refresh DDC display
})
