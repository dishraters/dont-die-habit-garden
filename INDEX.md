## ❓ FAQ

**Q: Can I deploy right now?**  
A: Yes! See [START_HERE.md](START_HERE.md)

**Q: Do I need Firebase credentials?**  
A: Not for MVP. App uses browser storage. Add later for persistence.

**Q: How long to deploy?**  
A: 2-5 minutes with Vercel

**Q: Will data persist?**  
A: Only during current session (localStorage). Phase 2 adds Firebase persistence.

A: See "Testing" section above
